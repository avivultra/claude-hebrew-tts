למה זה מגניב

שלושה דברים ששווים את זה

🎙️

קול נוירלי אמיתי

לא קול רובוטי מהשנות ה-90. Avri (גברי) ו-Hila (נשי) — אותם קולות של Microsoft Azure Neural, כמו ב-Edge Reader. שומעים הגייה טבעית, אינטונציה, שמות בעברית שנאמרים נכון.

🔓

חינם לחלוטין

אין מפתח API, אין הרשמה, אין מכסה חודשית. edge-tts משתמש ב־endpoint הציבורי של Microsoft — אותו אחד שהדפדפן Edge משתמש בו בתוכנה Read Aloud. pip install אחד ונגמר.

⚡

Skill = אוטומטי

לא פקודה שצריך לזכור. זה Skill של Claude Code — קלוד מזהה לבד ביטויים כמו "תקריא", "בקול", "הקרא לי" ומפעיל את הכלי. אפס תחזוקה.

איך מבקשים

תגיד משהו מהאלה וזה יעבוד

הסקיל מזהה כמה עשרות ניסוחים בעברית ובאנגלית. לא צריך פורמט קבוע — רק להגיד.
